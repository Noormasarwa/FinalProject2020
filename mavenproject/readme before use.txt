to lunch the project you need to : 

1- install on your phone GSM modem and GSM modem helper tool go to the smssend class and pu the user and the password and the ip for gsmmodem 
you can download it from google play or github https://github.com/sadiqodho/GSM-Helper-Tool/tree/master/download-app
2- install node.js on your computer 
3- after installing node.js open server file and open CMD and write install npm and run the server 
4- after that you need to install ngrok on your computer that will give you a TEMP Domain for 8 hours and turn it on (port 3030)
5- goto OPENBROWSER CLASS in the project set the URL to the URL that gave you on the ngrok.
6- go to the project file run the server interface 
7- run the server.
8- run the client interface. 
9- now its ready for you enjoy.
